# LeafletForBlazor-NuGet
 Using LeafletForBlazor NuGet Package

 - Map tooltips scaling. Configured using improved format:

     "tooltip":{
   
     ...
   
       "scaling": {
       
          "start_with": 16,

          "stop_with": 18
        
          }
          
       }

 - Map GeoJSON layer scaling. Configured using improved format:

   "symbology":{
   ...
       "scaling": {
        "start_with": 16,
        "stop_with": 18
        }
       }
